# 十二生肖贪吃蛇游戏

这是一个具有中国风特色的贪吃蛇网页游戏，以十二生肖为主题。

## 游戏特点

- 蛇头使用龙的元素表示
- 食物使用其他十一生肖表示
- 中国风的界面设计和色彩
- 根据难度级别生成不同的中国风障碍物
- 实时统计收集到的生肖数量

## 游戏控制

- 使用WASD键控制蛇的移动方向
- 可以通过滑块调整游戏速度
- 可以选择三种难度级别（简单、中等、困难）

## 如何开始

只需在浏览器中打开index.html文件即可开始游戏。
